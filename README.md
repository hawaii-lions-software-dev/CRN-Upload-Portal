
# Getting Started with Vite

This project uses [Vite](https://vitejs.dev/) for fast development and build tooling.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It bundles your code and optimizes the build for best performance.

### `npm run preview`

Serves the production build from the `dist` folder locally for previewing.
